欧阳克

编辑 锁定 讨论 上传视频
金庸武侠小说《射雕英雄传》中的反派角色，面目俊雅，英气逼人。
欧阳克是西毒欧阳锋的私生儿子，名义上的侄儿。白驼山少主。
他生性好色，拥有众多姬妾，受完颜洪烈重金聘请来盗取《武穆遗书》，曾受到江南七怪、洪七公等人的攻击。
在荒岛因要杀害洪七公而被黄蓉使计引开用大岩石压断双腿 [1]  ，后因在牛家村曲家酒馆调戏穆念慈和程瑶迦，被杨康偷袭刺穿肚腹杀死。

欧阳克人物经历

编辑
欧阳克漫画形象（李志清版射雕）(15张)
少年时，在云南大雪山中故意挑衅“农夫”武三通 [3]  ，双方打成平手。
青年时期，欧阳克受赵王完颜洪烈邀聘前往北京，途中抢掠良家女子和江南六怪撞见，被江南六怪打伤后逃走。
在赵王府，一出场便以高强功力震开王处一与沙通天的对掌，令二人大吃一惊。
随后比试武艺时，随手将二十只筷子撒成四个梅花形，令全场高手震惊 [4]  。在赵王府与完颜洪烈等商议武穆遗书的事情，后遇见黄蓉盗药，与黄蓉比试时，被黄蓉用计谋僵住。追捕黄蓉时遇见梅超风，被梅超风撕裂袍子，后与沙通天等人一起围攻梅超风 [5]  与江南六怪，随后协助完颜洪烈追赶杨铁心与包惜弱二人，遇全真教马钰与丘处机，几人与全真教定八月比武之约 [6]  。
在京东西路一带牧蛇，偶遇郭靖、黄蓉及洪七公三人，被洪七公威势所慑，悻悻离去 [7]  。
李解版欧阳克(10张)
在苏州之北三十里的一座荒山之中捉住穆念慈，强加戏弄，还摆蛇阵逼迫梅超风交出九阴真经，准备去献给叔父欧阳锋被黄药师以箫声击退 [8]  。
宝应县郊外刘氏宗祠，将穆念慈藏于空棺之中；掳劫程瑶迦，轻松胜过丐帮众人，后与郭靖交手，将胜之际洪七公赶到，洪七公将降龙十八掌剩余三掌教与郭靖，再次与郭靖交手，两人不分胜负的情况下，欧阳克暴露了所有武功底子，使出灵蛇拳打败郭靖；后与黄蓉比武遭到暗算，被洪七公救下后匆匆离去 [9]  。
宝应县，在路旁与杨康交谈后投东而去 [10]  。
欧阳克请叔叔欧阳锋向黄药师求亲，郭靖来到之时偷袭郭靖反被郭靖打断肋骨。
各个版本欧阳克(25张)
黄药师出三道试题考验欧阳克与郭靖二人。第一题比武被郭靖以蒙古摔跤之技反摔落地落败。第二题比试音律，黄药师吹奏起了碧海潮生曲，欧阳克内力不够几乎失去心智，黄药师以“比试音律而非内力”为由，偏袒欧阳克，欧阳克获胜。第三题比试背诵九阴真经下篇，郭靖因被周伯通诓骗捉弄早已经背熟了九阴真经全文，因此落败 [11]  。
欧阳克想留在岛上寻求其他接近黄蓉的机会，提出想向黄药师学习五行八卦，黄药师借桃花岛总图给其观阅 [12]  。
归途船上，与欧阳锋共同设毒计威逼郭靖写下九阴真经 [13]  ，后欲放火烧船，反是自己遭遇危险，被郭靖打断了右臂，与黄蓉、洪七公流落至荒岛。
岛上，欧阳克遭黄蓉偷袭受伤，在山洞里擒住黄蓉，被洪七公吓退；黄蓉使计将欧阳克拖入海中，欧阳克在水中抱起大石头在海底行走，走到海岸脱离危险；在欧阳克欲害洪七公之时，黄蓉将其骗入陷阱，欧阳克闪避不及，幸得赶来的欧阳锋相助而逃得性命，但双腿被从天而降的大石彻底压断 [1]  。
修庆版欧阳克(18张)
被救出后，与欧阳锋偷走黄蓉等人扎好的木筏抢先离岛，因黄蓉割断绳索，木筏在海上四分五裂，二人飘于大海，被路过的洪七公救上，发现了完颜洪烈的大船 [14]  。
随后被欧阳锋安置在曲灵风的酒店养伤。在侯通海等人被吓跑时，因腿折被落下 [15]  。
晚上欧阳克听到密室里面动静。翌日攻进密室未果，反而被郭靖、黄蓉打退，转而调戏程瑶迦与穆念慈，被杨康发现。
杨康不动声色，与欧阳克一起饮酒，乘其不备，用铁枪头刺入他的腹部将其杀死，欧阳克的尸体被他埋于废园 [2]  。随后又被黄药师拿回其身上的桃花岛总图 [16]  。
版本差异
刘智扬版欧阳克(4张)
连载版全书欧阳克无名，称其为欧阳公子，身份为白驼山山主；三联版开始加名字为欧阳克，身份变更为白驼山少主 [4]  。

欧阳克人物评价

编辑
覃贤茂：欧阳克色胆包天，可发一笑，拼死要作花下死的风流之鬼，求仁得仁，离死也差不多了 [17]  。
黄玉郎：欧阳克身为西毒侄儿，武功才学俱为晚一辈中的首选 [18]  ；俗语有云：色字头上一把刀，又谓：万恶淫为首，证诸欧阳克的下场，确实是个最佳的反面教材 [19]  。

欧阳克外貌描写

编辑
欧阳克（桃花岛）
1. 只见那人一身白衣，轻裘缓带，神态甚是潇洒，看来三十五六岁年纪，双目斜飞，面目俊雅，却又英气逼人，身上服饰打扮，俨然是一位富贵王孙。
2. 驱蛇人将蛇队分列东西，中间留出一条通路，数十名白衣女子手持红纱宫灯，姗姗而至，相隔数丈，两人缓步走来，先一人身穿白缎子金线绣花的长袍，手持折扇，正是欧阳克。
3. 黄蓉侧头去看父亲，见他望着二人呆呆出神，脸上神情也很奇特，只有欧阳克却不住的向她眉目传情，手中折扇轻挥，显得十分的倜傥风流。
4. 欧阳克左足一点，跃上竹亭，他有意卖弄轻功，轻飘飘的在亭角上一立，白袍在风中微微摆动。

欧阳克武功绝学

编辑
瞬息千里：白驼山庄家传的上乘轻功，身形飘忽，斗然间已飘到了对手身旁。
神驼雪山掌：白驼山庄家传掌法，变幻莫测，以掌进攻。
灵蛇拳法：从蛇身上悟出来的武学招式，手臂犹似忽然没了骨头，如变成了一根软鞭，打出后能在空中任意拐弯 [9]  。
